# API Stand Tests

Este proyecto contiene pruebas automatizadas para la API de Urban Grocers.

## Descripción
Pruebas para validar la creación de usuarios con diferentes parámetros del campo firstName.

## Estructura del proyecto
- `create_user_test.py` - Archivo principal con las pruebas
- `sender_stand_request.py` - Funciones para enviar solicitudes HTTP
- `data.py` - Datos de prueba
